## Enhanced Vanilla Modpacks

These modpacks were exported using Serbakon's configuration files. Keybindings or other settings may differ from the Minecraft defaults — expect some differences after importing.

### Resource Pack Order (Top → Bottom)

Place resource packs in the launcher/resource-pack list in the order below. The pack at the top has the highest priority.

1. QRAFTY's CAPITALIZED FONT
2. ENCHANT ICONS
3. TOOLTIPS-STYLIZED
4. FA+ Addons (any order)
5. FA+ All_Extensions
6. Fresh Animations
7. Remaining texture packs

Notes:
- FA+ Addons can be stacked in any order between themselves, but they should remain above FA+ All_Extensions.
- FA+ All_Extensions should be placed ABOVE Fresh Animations
- If visuals look wrong, try toggling specific packs off to find conflicts.

### Installation Tips
- Use the Minecraft launcher or your modpack manager to install resource packs in the order shown above.
- After importing, check `Options → Controls` to review keybindings — Serbakon's configs may override defaults.

### Troubleshooting
- Missing or broken textures: ensure all listed resource packs are enabled and ordered correctly.
- Conflicting features: disable FA+ Addons one-by-one to isolate issues.
